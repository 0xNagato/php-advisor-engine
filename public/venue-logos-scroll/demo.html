A responsive web component that displays venue logos in an animated, infinite scroll. Perfect for showcasing partner venues on any website.

Dynamic API Example

Fetches real venue data from your Laravel API

<venue-logos-scroll api-endpoint="/api/venue-logos" refresh-interval="300" cache-time="600000" loading-text="Loading venues from PRIMA..." pause-on-hover> </venue-logos-scroll>

Static Data Example (for comparison)

Uses hardcoded data instead of API

<venue-logos-scroll first-row='[{"id":1,"name":"Venue Name","logo_path":"https://..."}]' second-row='[{"id":2,"name":"Another Venue","logo_path":"https://..."}]' pause-on-hover> </venue-logos-scroll>

With Placeholders

Shows venue names when logos are not available

<venue-logos-scroll first-row='[{"id":1,"name":"Restaurant","logo_path":"..."},{"id":2,"name":"Bar","logo_path":""}]' second-row='[{"id":3,"name":"Club","logo_path":"..."}]' pause-on-hover show-placeholders> </venue-logos-scroll>

Fast Animation

Faster scrolling animation

<venue-logos-scroll first-row='[{"id":1,"name":"Venue","logo_path":"..."}]' second-row='[{"id":2,"name":"Another","logo_path":"..."}]' speed="fast" pause-on-hover> </venue-logos-scroll>

Same Direction Scrolling

Both rows scroll in the same direction

<venue-logos-scroll first-row='[{"id":1,"name":"Venue","logo_path":"..."}]' second-row='[{"id":2,"name":"Another","logo_path":"..."}]' direction="same" pause-on-hover> </venue-logos-scroll>

Custom Duration

Custom animation duration in seconds

<venue-logos-scroll first-row='[{"id":1,"name":"Venue","logo_path":"..."}]' second-row='[{"id":2,"name":"Another","logo_path":"..."}]' duration="50" pause-on-hover> </venue-logos-scroll>

Dark Theme

Dark mode styling

<venue-logos-scroll class="dark" first-row='[{"id":1,"name":"Venue","logo_path":"..."}]' second-row='[{"id":2,"name":"Another","logo_path":"..."}]' pause-on-hover> </venue-logos-scroll>
